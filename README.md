## 環境構築手順(Mac版)　　

[1]　仮想環境作成(venv)　　
```
$ python -m venv "仮想環境名"　　
```

[2]　仮想環境をアクティブ　　
```
$ . "仮想環境名"/bin/activate　　
```

[3] 各パッケージのインストール　　
https://github.com/noanoachan/AutoOperat_Mercari/blob/main/venv_AutoOperat_Mercari/requirements.txt　　
```
$ pip install -r requirements.txt
```

## ビルド手順　　
[1]　仮想環境内に srcフォルダを配置　　

[2]　任意のテキストエディタでビルド
